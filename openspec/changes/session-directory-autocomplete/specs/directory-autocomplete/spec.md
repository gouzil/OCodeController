## 新增需求

### 需求：输入时自动触发目录搜索

系统在用户输入工作目录路径时必须自动触发目录搜索并显示下拉提示。

#### 场景：用户输入非空路径
- **当** 用户在目录输入框中输入非空文字时
- **则** 系统将 `showPathPicker` 设置为 `true`
- **并且** 以 400ms 防抖延迟触发 `schedulePathSearch`
- **并且** 在搜索完成后显示匹配的下拉列表

#### 场景：用户清空输入框
- **当** 用户清空输入框（内容为空）时
- **则** 系统将 `showPathPicker` 设置为 `true`
- **并且** 调用 `listDirectory('/')` 显示根目录内容
- **并且** 同时显示"最近使用"列表

#### 场景：搜索进行中
- **当** `isSearchingPath` 为 `true` 时
- **则** 下拉列表中显示加载指示器

#### 场景：搜索完成有结果
- **当** 搜索完成且 `pathSearchResults` 非空时
- **则** 下拉列表中显示最多 8 条目录结果

#### 场景：搜索完成无结果
- **当** 搜索完成但 `pathSearchResults` 为空时
- **则** 下拉列表仅显示"最近使用"（如有）

### 需求：焦点时的下拉行为

系统在输入框获得焦点时必须正确初始化下拉列表内容。

#### 场景：输入框有内容时获得焦点
- **当** 输入框获得焦点且 `directory` 非空时
- **则** 系统触发 `schedulePathSearch` 搜索当前内容
- **并且** 下拉列表显示搜索结果

#### 场景：输入框为空时获得焦点
- **当** 输入框获得焦点且 `directory` 为空时
- **则** 系统调用 `listDirectory('/')` 列出根目录
- **并且** 下拉列表显示根目录子目录和"最近使用"记录

### 需求：列表项选择后自动填充

系统在用户点击下拉列表中的目录项后必须自动填充路径并收起列表。

#### 场景：用户点击搜索结果中的目录
- **当** 用户点击下拉列表中的目录项时
- **则** 系统将 `directory` 设置为该目录的完整路径
- **并且** 收起下拉列表（`showPathPicker = false`）
- **并且** 将该路径添加到最近使用记录

#### 场景：用户点击最近使用记录
- **当** 用户点击下拉列表中的最近使用路径时
- **则** 系统将 `directory` 设置为该路径
- **并且** 收起下拉列表
